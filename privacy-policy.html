Privacy Policy

Effective Date: 2025-01-12

Pretty JSON Viewer does not collect, store, or share any personal information from users. All operations and JSON data processing occur locally within your browser, and no data is transmitted to external servers or third-party services.

Data Collection and Use

  • Data Collected: None
  • Any JSON input provided by the user is processed entirely within the browser and is neither stored nor transmitted externally.

Permissions

  • clipboardRead: This permission is requested to allow the extension to read JSON data directly from the user's clipboard for convenience. The clipboard data is not stored or shared and is used solely within the context of the extension.
  • host_permissions (<all_urls>): This permission is included proactively for potential future features but is not actively used by the extension as it does not interact with external URLs or websites.

Changes to the Privacy Policy

This Privacy Policy may be updated over time. Users are advised to check the extension's Chrome Web Store page for the latest version.
